## Projetos em destaque

### Estudou — Plataforma de Organização de Estudos

Aplicação web desenvolvida com **Django** para organização de estudos.

O sistema permite que cada usuário cadastre suas próprias matérias, organize módulos e tópicos, acompanhe o progresso dos estudos e mantenha seus dados isolados dos demais usuários.

**Principais funcionalidades:**

- Cadastro de usuários
- Login e logout
- Perfil de usuário
- Alteração de senha
- CRUD de matérias, módulos e tópicos
- Dashboard com resumo dos estudos
- Cálculo de progresso geral e por matéria
- Busca e filtros
- Isolamento de dados por usuário
- Testes automatizados

**Tecnologias utilizadas:**

- Python
- Django
- SQLite
- HTML
- CSS
- Git
- GitHub

**Repositório:**

### Gerenciador de Tarefas Full Stack

Projeto full stack desenvolvido com **FastAPI**, **React**, **SQLite** e **Docker Compose**.

A aplicação permite cadastrar, listar, editar, concluir, reabrir e remover tarefas, com persistência dos dados em banco SQLite e execução completa via Docker Compose.

**Principais funcionalidades:**

- Cadastro de tarefas
- Listagem de tarefas
- Edição de tarefas
- Conclusão e reabertura de tarefas
- Remoção de tarefas
- Integração entre front-end e back-end
- Persistência em banco de dados
- Execução com Docker Compose

**Tecnologias utilizadas:**

- Python
- FastAPI
- SQLModel
- SQLite
- React
- Vite
- Docker
- Docker Compose

**Repositório:**  
https://github.com/ovictorvale/api-tarefas-fastapi
